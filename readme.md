## Game Agnostic Tools For Developing Mods and Games for Luanti Engine

### *Building immersive voxel adventures*

---

## Overview

Voxrame is a comprehensive modpack that provides essential tools and systems for creating immersive voxel-based games and mods on the Luanti Engine (formerly Minetest). It offers a collection of modular components that can be used independently or together to build rich gaming experiences.

## Installation

*Instructions will be available soon on [voxrame.dev](https://voxrame.dev)...*

To start right now:
```bash
git submodule add https://github.com/Voxrame/Voxrame.git mods/Voxrame
```
